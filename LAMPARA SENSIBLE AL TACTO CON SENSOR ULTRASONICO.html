Lampara sensible al tacto con sensor ultrasonico

Este proyecto tiene como finalidad fundamentar y conllevar distintos mecanismos de circuitos simples, mediante la implementación del uso de arduino para la realización de este proyecto, uno de las principales generalidades de este proyecto es conllevar y inculcar el uso de diferentes medios para la realización de circuitos simples, siempre y cuando sea supervisado su proceso.

Marco Teórico

Que es una lámpara sensible al tacto

La capacidad es una medida que indica la capacidad de una carga eléctrica que un campo eléctrico que un cuerpo puede almacenar la Biblioteca comprueba dos terminales de arduino uno es emisor y otro es receptor mide el tiempo que tarda en alcanzar el mismo estado . Estos terminales se conectaran a un objeto metálico su cuerpo absorberá alguna carga . Haciendo que transcurra mas tiempo para que los dos terminales tengan el mismo estado.


  1. El uso del arduino para la programación de los sensores es fundamental debido a que facilita su utilidad y conlleva a la finalidad del proyecto

  2. El uso del prothoboard como medio conductor de energía que permite el flujo de corrientes eléctricas para la utilidad de este proyecto

  3. El uso de las resistencias para la estabilización del flujo de corriente eléctrica.

Materiales y equipos

  1. Arduino uno

  3. Placa protoboard

  4. Diodo LED

  5. Cables conectores macho

  7. Sensor ultrasonico

Precauciones o riesgos de la práctica

Este proyecto tiene un indice de riesgos muy bajos, basicamente el riesgo mas alto es al momento de la conexión de los componentes a la protoboard, debido que pueden quemarse o hacer corto.

Resultados y conclusiones de la práctica

Como conclusiones para este proyecto cabe recalcar que su función de lamapara sensible al tacto fue un éxito, también se puede evidenciar que durante la practica de este proyecto, se pudo ver una dificultad, la cual fue el montaje del código debido a los parámetros a indicar.
